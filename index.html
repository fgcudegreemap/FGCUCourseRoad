This tool is not endorsed by Florida Gulf Coast University and is only intended as a reference to supplement the advising process. Please feel free to use it to understand the requirements of your degree and whether prerequisite(s) for your intended course schedule are satisfied.

To use this tool, select your intended major and add classes by their class code to the desired semester or time period in your course schedule. If you want to "hide" long lists of extra classes that fulfill a particular requirement after you've completed it, select that option as well. The system will automatically update to show you what requirements and prerequisite(s) you have met or still require. Please note, the system currently only accepts courses listed in the FGCU Course Catalog for the Fall 2016 semester. Courses must be entered in the format "ABC XXXX" or "ABC XXXXX" without quotations.

Please note, this current version of the Degree Mapper does not support corequisite courses. You can enter them, but there's still some work to be done to register that your class requires a corequisite. Also, if you have AP, transfer, or other non-coursework credits, please enter them as an equivalent FGCU class, we aren't able to support those kinds of inputs yet. Beyond that, if you have questions, if any errors come up, or a course is missing, etcetera, please email Jackson Graves at jrgraves@mit.edu with feedback to implement a fix.
